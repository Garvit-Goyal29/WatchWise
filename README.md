# WatchWise – AI Movie Recommendation System
*Watch what you feel — powered by AI*

---

## Overview

Choosing what to watch can be frustrating and time-consuming.
**WatchWise** solves this by using AI to recommend movies based on your mood, preferred language, and industry — all within seconds.

---

## How It Works

1. 🧠 User describes their mood
2. 🤖 AI processes input using Groq / Gemini
3. 🎬 Backend fetches movie data via OMDB API
4. 📊 Results are displayed with full movie details

## 📦 Installation

```bash
git clone https://github.com/Garvit-Goyal29/WatchWise.git
cd WatchWise
npm install
```

---

## ▶️ Run the Project

### Backend

```bash
npm start
```

### Frontend

```bash
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file in backend:

```env
GROQ_API_KEY=your_groq_api_key
GEMINI_API_KEY=your_gemini_api_key
OMDB_API_KEY=your_omdb_api_key
```
